0x00 前言

URLDNS是java反序列化中一个比较简单的链，所以在学习CommonsCollections的利用链之前，可以先从URLDNS看起。

0x01 什么是URLDNS

URLDNS是ysoserial中的一个利用链名字，不过URLDNS并不能执行命令，只能发送DNS请求。

但是很适合我们在检测反序列化漏洞时使用：

  • 使用Java内置的构造类，对第三方库没有依赖
  • 在目标没有回显的时候，可以利用DNS请求得知是否存在反序列化漏洞
